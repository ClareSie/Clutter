# 如何寫書  
1.選題目，找素材，訂架構，寫初稿，持續修正  
2.思考用戶的痛點及能夠用戶帶來什麼價值  
3.分析為什麼讀者願意閱讀、分享你的書  
4.站在讀者立場思考，篩選吸引讀者的題材   
5.好標題
6.內容連貫  
7.能先說服自己
