# 基礎
Python Interface: abc(Abstract Base Class)  
盡量不要new, 用abc或帶入參數降低耦合  
繼承Class: class 子(class 父):  
介面:  
```
from abc import ABC, abstractmethod  
class 父(abc):
    @abstractmethod
    def 抽象方法(self):
        pass  
class 子(父):
    def 實作抽象方法(self):
        ...
```
